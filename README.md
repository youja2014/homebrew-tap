# homebrew-tap

Homebrew tap for personal CLI tools.

## Usage

```bash
brew tap youja2014/tap
brew install smoothline
```

Or in a single line:

```bash
brew install youja2014/tap/smoothline
```

## Available formulae

| Formula | Description | Upstream |
|---|---|---|
| [smoothline](Formula/smoothline.rb) | Smooth, gradient statusline for Claude Code | [youja2014/smoothline](https://github.com/youja2014/smoothline) |
